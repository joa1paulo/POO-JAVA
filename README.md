## 📖 Sobre a Disciplina
A Programação Orientada a Objetos é um paradigma de programação baseado no conceito de "objetos", que podem conter dados na forma de campos e códigos na forma de procedimentos. Durante o curso, exploramos como modelar problemas do mundo real em estruturas de classes.
